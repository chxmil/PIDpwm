🤖 AI-Enhanced Haptic Gripper: Adaptive PID ControlResearcher: Chamil Ahlee | Computer Engineering & AI, Walailak Universityโปรเจกต์นี้เป็นการพัฒนาระบบควบคุมกริปเปอร์หุ่นยนต์แบบอัจฉริยะ โดยใช้การทำนายแรงบิด (Force Inference) จากโมเดล CNN-LSTM ร่วมกับตัวควบคุม PID (Proportional-Integral-Derivative) เพื่อให้กริปเปอร์สามารถรักษาแรงบิดได้คงที่และนุ่มนวลตามเป้าหมาย (Set-point) แม้จะเปลี่ยนวัสดุหรือตัวเซนเซอร์🚀 Key FeaturesAI-Adaptive PID Control: ใช้ AI ในการทำนายแรงบิด (Tactile Sensing) แบบ Real-time เพื่อเป็น Feedback ให้กับ PID ControllerDynamic Baseline Shifting: ระบบ Calibrate และปรับจูนค่าเริ่มต้นของเซนเซอร์โดยอัตโนมัติ เพื่อแก้ปัญหา Sensor Drift และรองรับเซนเซอร์ที่ต่างรุ่นกันCross-Hardware Compatibility: ใช้เทคนิค Gain Scaling (เช่น Gain 0.08 สำหรับเซนเซอร์ใหม่) เพื่อให้โมเดลเดิมทำงานร่วมกับฮาร์ดแวร์ใหม่ได้ทันทีโดยไม่ต้องเทรนใหม่1.8 Hz Deterministic Loop: ระบบควบคุมถูกออกแบบให้ทำงานที่ความถี่ 1.8 Hz เพื่อให้สอดคล้องกับการประมวลผลของโมเดล CNN-LSTM📂 Project StructureApp.py: ส่วนควบคุมหลัก (Main Entry Point) จัดการ Serial Communication, Heartbeat และคำสั่ง InteractiveModelInclude.py: ระบบประมวลผล AI และคำนวณ PID Logic รวมถึงขั้นตอนการทำ Baseline ShiftingAnalysis2ndSensor.ipynb: ท่อส่งข้อมูล (Data Pipeline) สำหรับทำ Feature Engineering และเตรียม Dataset เพื่อเทรนโมเดล/Model: โฟลเดอร์เก็บไฟล์โมเดล .keras และ Scalers .pkl🛠️ System ArchitectureHardware: ESP32-S3 ร่วมกับเซนเซอร์ Piezoresistive และมอเตอร์ที่ควบคุมผ่าน PWMInference Engine: โมเดล CNN-LSTM รับค่า shifted_cond (Conductance) และ is_press เพื่อทายค่า force_nControl Loop: * Input: Target Force (N)Process: PID Calculation $\to$ PWM Scaling $\to$ Low-pass FilteringOutput: PWM Command ส่งกลับไปยัง ESP32📝 Setup & Usage1.
